## 작성 팁

### ✅ 좋은 보고서의 특징:
1. **정량적**: 숫자, 퍼센트, 날짜 포함
2. **구체적**: "잘 진행" 대신 "API 12개 중 10개 완료"
3. **명확한 이슈**: 문제의 원인과 영향 명시
4. **실행 가능한 계획**: "열심히 하겠습니다" 대신 구체적 액션
5. **솔직한 요청**: 도움 필요 시 명확히 표현

### ❌ 피해야 할 표현:
- "잘 진행되고 있습니다" → 무엇이 얼마나?
- "조금 어려움" → 어떤 어려움? 얼마나 심각?
- "최선을 다하겠습니다" → 구체적으로 무엇을?
- "검토 중입니다" → 언제까지? 누구와?
